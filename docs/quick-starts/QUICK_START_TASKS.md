# Quick Start: Task Management System

This guide will help you get started with the volunteer task management system.

## Overview

The task management system provides:

- Task creation and assignment
- Calendar view of scheduled tasks
- Task status tracking
- Volunteer coordination
- Recurring task support
- Task notifications

## Prerequisites

- Volunteer or Admin role
- Logged in to the application
- Animals and territories configured

## Accessing Tasks

1. Navigate to `/tasks` from the dashboard
2. Or click "Tasks" in the navigation menu
3. You'll see the task management interface

## Task Interface Overview

### Calendar View

- Monthly calendar with task indicators
- Color-coded by task type
- Click dates to view tasks
- Navigate between months

### Task List

- Shows tasks for selected date
- Filtered by status
- Quick actions available
- Real-time updates

### Quick Actions Panel

- Create new task
- Filter by status
- Filter by type
- Export tasks

## Creating a Task

### Step 1: Open Create Form

Click the "Create Task" button or "+" icon

### Step 2: Fill in Task Details

**Required Fields:**

- **Title**: Brief task description
- **Type**: Feeding, Medical, Cleaning, Monitoring, Maintenance, or Event
- **Date**: When task should be completed
- **Time**: Specific time for task
- **Location**: Where task takes place

**Optional Fields:**

- **Description**: Detailed instructions
- **Assigned To**: Volunteer responsible
- **Related Animal**: Associated animal (if applicable)
- **Related Territory**: Associated territory (if applicable)
- **Priority**: Low, Medium, High, or Urgent
- **Estimated Duration**: How long task will take
- **Recurring**: Set up repeating tasks

### Step 3: Set Priority

Choose priority level:

- **Low**: Non-urgent routine tasks
- **Medium**: Standard daily tasks
- **High**: Important tasks requiring attention
- **Urgent**: Critical tasks needing immediate action

### Step 4: Assign Volunteer

- Select from dropdown of volunteers
- Leave unassigned for open tasks
- Assigned volunteer receives notification
- Can reassign later if needed

### Step 5: Configure Recurrence (Optional)

For recurring tasks:

1. Enable "Recurring Task" toggle
2. Select frequency: Daily, Weekly, Monthly
3. Set end date or number of occurrences
4. System creates task series automatically

### Step 6: Save

Click "Create Task" to save

## Managing Tasks

### Updating Task Status

**Mark as In Progress:**

1. Click task card
2. Click "Start Task" button
3. Status updates automatically
4. Notification sent to coordinators

**Mark as Completed:**

1. Click task card
2. Click "Complete Task" button
3. Add completion notes (optional)
4. Status updates automatically
5. Notification sent to assignee and coordinators

**Cancel Task:**

1. Click task card
2. Click "Cancel Task" button
3. Provide reason (optional)
4. Task marked as cancelled

### Editing Tasks

1. Click task card
2. Click "Edit" button
3. Update information
4. Click "Save Changes"
5. Notifications sent if assignment changed

### Deleting Tasks

1. Click task card
2. Click "Delete" button
3. Confirm deletion
4. Task permanently removed

## Task Notifications

### Automatic Notifications

**Task Assigned:**

- Sent when task is assigned to you
- Includes task details and due date
- Click to view task

**Task Reminder:**

- Sent 24 hours before due time
- Helps you prepare
- Includes location and instructions

**Task Completed:**

- Sent to coordinators
- Shows who completed task
- Includes completion notes

**Task Overdue:**

- Sent if task not completed by due time
- Reminder to complete or reschedule
- Escalated to coordinators

### Managing Notifications

- Configure in notification preferences
- Enable/disable task reminders
- Set reminder timing
- Choose delivery method (email/in-app)

## Recurring Tasks

### Creating Recurring Tasks

1. Enable "Recurring Task" toggle
2. Select frequency:
   - **Daily**: Every day
   - **Weekly**: Same day each week
   - **Monthly**: Same date each month
3. Set end condition:
   - End date
   - Number of occurrences
   - Never (ongoing)

### Managing Recurring Tasks

**Edit Series:**

- Changes apply to all future occurrences
- Past occurrences unchanged
- Option to update all or just future

**Edit Single Occurrence:**

- Changes apply to one instance only
- Series continues unchanged
- Useful for exceptions

**Cancel Series:**

- Stops creating future occurrences
- Past occurrences remain
- Can restart if needed

## Troubleshooting

### "Failed to create task"

- Check all required fields
- Verify date/time is valid
- Ensure location is specified
- Check network connection

### Task Not Appearing

- Check date filter
- Verify status filter
- Refresh the page
- Check task was saved

### Can't Assign Volunteer

- Verify volunteer exists
- Check volunteer role
- Ensure volunteer is active
- Contact admin if needed

### Notifications Not Received

- Check notification preferences
- Verify task has due date/time
- Ensure email is configured
- Review spam folder
